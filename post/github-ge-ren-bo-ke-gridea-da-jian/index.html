Github个人博客,Gridea搭建

2020-12-24

Gridea官方网站: https://gridea.dev/

1.建立github仓库
1）首先点击"+"好, 新建仓库

2）仓库名为 [用户名.github.io] ，后续就可以用这个仓库名作为博客访问域名, 仓库选择公开。

注意：输入的是你未来要访问的URL
3）选择仓库时候，需要新建目录，才能选择，要不会显示为none。


4）创建仓库后点击仓库的settings，拉到页面最端检查自己是否开启了github pages


到这里，就成功建立了内容为空的博客页面，访问域名是你的仓库名。
2.建立github token
点击右上角头像的settings，选择菜单最下面的developer settings

选择personal access tokens，并生成一个新的token

新建token时填完名称后把repo勾选起来，然后到点击生成
生成成功后，建议将你的令牌保存起来，根据提示，你的这个令牌关掉页面后你再也找不到了
3.配置Gridea

4.开启文章评论
点击链接https://github.com/settings/applications/new, 创建


创建完成后可将 Client ID 和 Client Secret 填到Gridea
